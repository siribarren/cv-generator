# CV Generator SERMALUC - GCP Ready

Proyecto completo con nuevo flujo de selección, construcción preliminar y aprobación de CVs.

Flujo:
1. Subir CVs + Requerimiento
2. Mostrar tabla con % match, puntos fuertes y débiles (selección con checkbox)
3. Construir CVs preliminares solo para candidatos seleccionados
4. Mostrar tabla con CVs preliminares y links para edición
5. Aprobar y generar PDFs finales

Optimizado para:
- React + Vite
- Node.js + Express + Puppeteer
- GCP Cloud Run + Cloud Build
